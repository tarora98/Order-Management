Below endpoint are used to get the desired result

1) Get all the Order
    Endpoint:- /orders

2) Get order as per the id
    Endpoint:- '/orders/:id

3) Get New Order as per the requirement
    method:- POST
    Endpoint:- /orders

4) Update an existing order
    method:- PUT
    Endpoint:- '/orders/:id'

5) Delete an order
    method:- Delete
    Endpoint:- /orders/:id

DataBase:- MYSQL
